Kebijakan Operator ini mengenai pemrosesan data pribadi (selanjutnya - Kebijakan) berlaku untuk semua informasi yang Operator dapat terima tentang pengunjung ke situs web. Situs 2. Konsep utama yang digunakan dalam Kebijakan Pemrosesan data pribadi secara otomatis - pemrosesan data pribadi menggunakan teknologi komputer; Pemblokiran data pribadi - penghentian sementara pemrosesan data pribadi (kecuali jika pemrosesan diperlukan untuk mengklarifikasi data pribadi); Situs Situs web - satu set materi grafis dan informasi, serta program komputer dan database, memastikan ketersediaannya di Internet pada alamat jaringan; Situs Sistem informasi data pribadi - satu set data pribadi yang terkandung dalam basis data, dan memastikan pemrosesan teknologi informasi dan sarana teknis mereka; Situs Anonimisasi data pribadi - tindakan, sebagai akibatnya tidak mungkin untuk menentukan tanpa menggunakan informasi tambahan kepemilikan data pribadi untuk Pengguna tertentu atau subjek data pribadi lainnya; Memproses data pribadi - setiap tindakan (operasi) atau serangkaian tindakan (operasi) yang dilakukan dengan menggunakan alat otomasi atau tanpa menggunakan alat tersebut dengan data pribadi, termasuk pengumpulan, perekaman, sistematisasi, akumulasi, penyimpanan, klarifikasi (memperbarui, mengubah), pengambilan, penggunaan , transfer (distribusi, penyediaan, akses), depersonalisasi, pemblokiran, penghapusan, penghancuran data pribadi; Situs Operator - badan negara, badan kota, badan hukum atau perorangan, secara mandiri atau bersama-sama dengan orang lain, mengatur dan (atau) memproses data pribadi, serta menentukan tujuan pemrosesan data pribadi, komposisi data pribadi yang akan diproses, tindakan (operasi) yang dilakukan dengan data pribadi; Data pribadi - setiap informasi yang berkaitan langsung atau tidak langsung dengan Pengguna Situs Web yang ditentukan atau ditentukan; Pengguna - pengunjung situs web apa pun; Penyediaan data pribadi - tindakan yang bertujuan untuk mengungkapkan data pribadi kepada orang tertentu atau lingkaran orang tertentu; Distribusi data pribadi - setiap tindakan yang bertujuan untuk mengungkapkan data pribadi ke lingkaran orang yang tidak ditentukan (transfer data pribadi) atau untuk membiasakan diri dengan data pribadi dari lingkaran orang yang tidak terbatas, termasuk mengungkapkan data pribadi di media, memposting informasi dan jaringan telekomunikasi atau menyediakan akses data pribadi dengan cara lain apa pun; Transfer data pribadi lintas batas - transfer data pribadi ke wilayah negara asing ke otoritas negara asing, individu asing atau badan hukum asing; Penghancuran data pribadi - setiap tindakan sebagai akibat dari mana data pribadi dihancurkan secara permanen dengan ketidakmampuan untuk lebih lanjut mengembalikan konten data pribadi dalam sistem informasi data pribadi dan (atau) sebagai akibatnya penghancuran materi pembawa data pribadi terhadap data pribadi .
